Интеллектуальная диалоговая система, которая использует Retrieval-Augmented Generation (RAG) для симуляции экспертных консультаций в различных профессиональных областях.

## Техническая реализация

- **RAG-пайплайн**: Пользовательская реализация для контекстно-зависимых ответов
- **Поддержка нескольких доменов**: Специализированные базы знаний для различных экспертных областей
- **Масштабируемое развертывание**: Контейнеризованная архитектура с Docker

## Случаи использования

Идеально подходит для образовательных целей, первичных консультаций и изучения различных профессиональных перспектив по сложным темам.
