Комплексная EdTech-платформа, которая объединяет машинное обучение с современными веб-технологиями для обеспечения интерактивного опыта обучения для студентов программирования.

## Ключевые возможности

- **Обнаружение плагиата на основе ML**: Анализ кода в реальном времени с использованием пользовательских ML-моделей
- **Система подсказок на основе ИИ**: Контекстно-зависимые предложения на основе LLM
- **Масштабируемая архитектура**: Микросервисы с Kafka для потоковой передачи событий
- **Современный стек**: React/Next.js фронтенд, FastAPI бэкенд

## Влияние

Платформа активно используется образовательными организациями, помогая сотням студентов улучшать свои навыки программирования через интерактивные упражнения и интеллектуальную обратную связь.
